Precedência:

Obs.: Não receber mensagens de erro não significa que seu programa está certo. Pode não ter erro sintático mas a execução pode estar errada.

Parênteses muda a ordem de precedência.

        > 5 + 3 / 2
        6.5
        > (5 + 3) / 2
        4
    
        () -> Parênteses

        ** -> Potência

        * / %  -> Multiplicação, Divisão e resto da divisão. (será feito quem aparecer primeiro da esquerda para a direita.)

        + -  -> por ultimo, Adição e subtração.

Atribuição Simples:

        var a = 5 + 3          8
        var b = a % 5          3
        var c = 5 * b ** 2     45
        var d = 10 - a / 2     6
        var e = 6 * 2 / d      2
        var f = b % e + 4 / e  3

Auto-atribuição:

        var n = 3                 era 3
        n = n + 4   ou   n += 4   agora é 7
        n = n - 5   ou   n -= 5   agora é 2
        n = n * 4   ou   n *= 4   agora é 8
        n = n / 2   ou   n /= 2   agora é 4
        n = n ** 2  ou   n **= 2  agora é 16
        n = n % 5   ou   n %= 5   agora é 1
    
        Simplificando: n += 4, só quando a mesma variável se repete.
    
        Exemplos:
    
        > var num = 8
        undefined
        > num
        8
        > num += 2
        10
        > num
        10
        > num %= 2
        0
        > num = 3
        3
        > num **= 2
        9
        > num
        9

Incremento:

        var x = 5                        5
        x = x + 1  ou  x += 1  ou x++    6
        x = x - 1  ou  x -= 1  ou x--    5
        incremento é uma simplificação maior x++ e x--
    
        > var n = 10
        undefined
        > n
        10
        > n ++ (incremento)
        10
        > n
        11
        > n -- (decremento)
        11
        > n
        10
        > ++n (pré-incremento)
        11
        > --n (pós-decremento)
        10
