Operadores

Operadores Relacionais

5 > 2.....true
7 < 4.....false
8 >= 8...true
9 <= 7...false
5 == 5...true
4 != 4....false

Exemplos:
Preço >= 200.50 O preço é maior ou igual a 200.50?
idade < 18 A idade é menor do que 18?
curso == 'JavaScript' O curso pe JavaScript?
n1 != n2 O primeiro número pe diferente do segundo?

Operadores de Identidade

5 == 5......true
5 == '5'.....true
5 === '5'...false (sinal de idêntico)
5 !== '5'....true (sinal de desigual restrito, são de tipos diferentes mas o valor interno é o mesmo.)

Operadores Lógicos

!........ negação
&&....conjunção
||........disjunção

se não(!) é true é false
se não(!) é false = true

true e(&&) true = true
true e(&&) false = false
false e(&&) true = false
false e(&&) false = false

true (ou)|| true = true
true (ou)|| false = true
false (ou)|| true = true
false (ou)|| false = false

obs.: vários operadores em uma mesma expressão:

primeiro vou resolver:
os operadores aritiméticos,
depois os relacionais,
depois os lógicos.

Se em uma expressão eu tiver &&(e) !(não) e ||(ou), a ordem de execussão é sempre, primeiro o !(não) depois o &&(e) depois o ||(ou).

exemplos:
idade >= 15 && idade <= 17 A idade está entre 15 e 17?
estado == 'RJ' || estado == 'SP' O estado é RJ ou SP?
Salário > 1500 && sexo != 'M' o salário é acima de 1500 e não é homem?

Precedência

  1. ( ), **, /, ...
  2. >, <, >=, ...
  3. !
  4. &&
  5. ||

Operadores ternários

teste (teste lógico) ? true (se for verdadeiro...) : false (se for falso...)

exemplo:
A média >= 7.0 ? (se for vdd) "Aprovado" : (Se for falso) "Reprovado"
